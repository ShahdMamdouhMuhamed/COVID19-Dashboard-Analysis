# COVID-19 Global Dashboard Analysis

## 📌 Project Overview

This project presents an interactive dashboard for analyzing the global spread and impact of COVID-19. The dashboard is built using **Dash by Plotly** and provides dynamic visualizations to explore cases, deaths, vaccinations, and other key indicators across countries.

---

## 🎯 Objectives

* Analyze COVID-19 trends over time
* Compare pandemic metrics across different countries
* Study the impact of vaccinations on cases and deaths
* Explore relationships between socio-economic factors and COVID-19 outcomes
* Build an interactive and user-friendly dashboard

---

## 📊 Dataset

* Source: Our World in Data
* File: `owid-covid-data.csv`
* Description: Contains global data on COVID-19 cases, deaths, vaccinations, testing, and socio-economic indicators

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Plotly
* Dash
* Google Colab

---

## 📈 Dashboard Features

* Multiple chart types:

  * Bar & Column Charts
  * Stacked & Clustered Charts
  * Scatter & Bubble Charts
  * Histogram, Box, and Violin Charts
  * Line & Area Charts
* Interactive elements:

  * Dropdown menus (country selection)
  * Sliders (date range)
  * Radio buttons / checklists
* Dynamic updates using Dash callbacks

---

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/COVID19-Dashboard-Analysis.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the dashboard:

```bash
python app.py
```

4. Open your browser and go to:

```
http://127.0.0.1:8050/
```
